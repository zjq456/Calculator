# 计算器 Calculator

这是一个简单的计算器，用typescript算法实现，支持两个数字的加减乘除，并支持负数、小数，并且判定若超过最大限制则输出 `Exceed the Limit`，如果遇到除 $0$，则输出 `Error`。

This is a simple calculator implemented using the typescript algorithm, supporting addition, subtraction, multiplication, and division of two numbers, as well as negative numbers and decimals. If the maximum limit is exceeded, it outputs `Exceed the Limit`, and if division by $0$ is encountered, it outputs `Error`.
